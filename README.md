# ModelingPointsWithODEs
Simulates point(s) on a continuous function that is effected by gravity, friction, etc. Uses Differential Equations to model this as a vector field using libraries such as Numpy and Matplotlib.

## What is required to run?
This is being run on <b>Python 3.12</b>, but some earlier versions & later versions ought to work. <br>
Ensure you have the following libraries:
- <b>Numpy</b>
- <b>Matplotlib</b>

## Why am I doing this?
After watching <b>3Blue1Brown</b>'s first video in their <i>Tourist's Guide to Differential Equations</i> series, where they modeled a pendulum using an ODE (Ordinary Differential Equation), I felt inspired. <br>

I wanted to come up with a problem that'd produce a similar looking vector field, but I didn't want to just do the same problem.
This is when I came up with the idea to have a continuous function on a 2d graph, and to have a point on the resulting line.
This point would be effected by gravity, friction, and later more. <br>

After producing some admittedly messy code, and interest from teachers, I decided I'd get to learning more python, and produce more readable code. <br>

And thus, this project was born.
